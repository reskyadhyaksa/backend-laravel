## Installation

##### 1. Clone the repository

```
git clone https://github.com/reskyadhyaksa/backend-laravel.git
```

##### 2. Select Directory
```
cd backend-laravel
```


##### 3. Install Depedency

```
composer install
```

##### 4. Copy env from env.example

```
cp .env.example .env
```

##### 5. Migrate the DB

```
php artisan migrate
```

##### 6. Run Backend

```
php artisan serve
````
