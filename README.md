## Installation

##### 1. Clone the repository

```
git clone https://github.com/reskyadhyaksa/backend-laravel.git
```

##### 2. Install Depedency

```
composer install
```

##### 3. Copy env from env.example

```
cp .env.example .env
```

##### 4. Migrate the DB

```
php artisan migrate
```

##### 5. Run Backend

```
php artisan serve
````
